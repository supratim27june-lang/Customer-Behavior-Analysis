# Customer-Behavior-Analysis
Data Analytics Project showcasing customer behavior analysis using Python, PostgreSQL and Power BI.
NOTE:- Since I was coding in Google Colab and my pgAdmin4 was hosted on localhost:5432, Colab was unable to connect to the localhost server, that is why there is an error message at the 2nd last cell. 
If the same is ran on jupyter notebook hosted in localhost, it will not give an error and there will be no need to create a separate processed CSV file. The SQL database can be connected to the notebook directly using the exact code used in Colab's 2nd last cell.
